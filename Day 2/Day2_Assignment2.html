-------------------------------------------

String Methods

-------------------------------------------

1) String Length

The length property returns the length of a string.

Example:

<script> var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"; console.log("String length is: ", txt.length); </script>

2) Finding a String in a String

The indexOf() method returns the index of (the position of) the first occurrence of a specified text in a string.

The lastIndexOf() method returns the index of the last occurrence of a specified text in a string

Example:

<script> var str = "Please locate where 'locate' occurs!"; var firstpos = str.indexOf("locate"); var lastpos = str.lastIndexOf("locate"); console.log(firstpos); console.log(lastpos); </script>

3) Searching for a String in a String

The search() method searches a string for a specified value and returns the position of the match.

Example:

<script> var str = "Please locate where 'locate' occurs!"; var pos = str.search("locate"); console.log(pos); </script>

4) slice()

slice() extracts a part of a string and returns the extracted part in a new string.

Example:

<script> var str = "Apple, Banana, Kiwi"; var res = str.slice(7, 13); console.log(res); </script>

5) substring()

substring() is similar to slice().

The difference is that substring() cannot accept negative indexes.

Example:

<script> var str = "Apple, Banana, Kiwi"; var res = str.substring(7, 13); console.log(res); </script>

6) substr()

substr() is similar to slice().

The difference is that the second parameter specifies the length of the extracted part.

Example:

<script> var str = "Apple, Banana, Kiwi"; var res = str.substr(7, 6); console.log(res); </script>

7) Replacing String Content

The replace() method replaces a specified value with another value in a string.

Example:

<p>Replace "Microsoft" with "Google" in the paragraph below:</p> <button onclick="myFunction()">Try it</button> <p id="demo">Please visit Microsoft!</p> <script> function myFunction() { var str = document.getElementById("demo").innerHTML; var txt = str.replace("Microsoft", "Google"); document.getElementById("demo").innerHTML = txt; } </script>

8) Converting to Upper and Lower Case

A string is converted to upper case with toUpperCase().

A string is converted to lower case with toLowerCase().

Example:

<script> var text1 = "Hello World!"; // String var text2 = text1.toUpperCase(); var text3 = text1.toLowerCase(); console.log("The UpperCase string is: ", text2); console.log("The LowerCase string is: ", text3); </script>

9) concat()

concat() joins two or more strings.

Example:

<script> var text1 = "Hello"; var text2 = "World"; var text3 = text1.concat(" ", text2); console.log(text3); </script>

10) trim()

The trim() method removes whitespace from both sides of a string.

Example:

<script> var str = " Hello World! "; alert(str.trim()); </script>

11) charAt()

The charAt() method returns the character at a specified index (position) in a string.

Example:

<script> var str = "HELLO WORLD"; console.log(str.charAt(0)); // returns H </script>

12) charCodeAt()

The charCodeAt() method returns the unicode of the character at a specified index in a string.

Example:

<script> var str = "HELLO WORLD"; console.log(str.charCodeAt(0)); // returns 72 </script>

13) Converting a String to an Array

A string can be converted to an array with the split() method.

Example:

<script> var str = "a,b,c,d,e,f"; var arr = str.split(","); console.log(arr); </script>

-------------------------------------------

Array Methods

-------------------------------------------

1) Converting Arrays to Strings

The JavaScript method toString() converts an array to a string of (comma separated) array values.

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; console.log(fruits.toString()); </script>

2) pop()

The pop() method removes the last element from an array.

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.pop(); // Removes the last element ("Mango") from fruits console.log(fruits); </script>

3) push()

The push() method adds a new element to an array (at the end).

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.push("Kiwi"); // Adds a new element ("Kiwi") to fruits console.log(fruits); </script>

4) Shifting Elements

The shift() method removes the first array element and "shifts" all other elements to a lower index.

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.shift(); // Removes the first element "Banana" from fruits console.log(fruits); fruits.unshift("Lemon"); // Adds a new element "Lemon" to fruits console.log(fruits); </script>

5) Deleting Elements

Elements can be deleted by using the JavaScript operator delete.

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; delete fruits[0]; // Changes the first element in fruits to undefined console.log(fruits); </script>

6) Splicing an Array

The splice() method can be used to add new items to an array.

The first parameter (2) defines the position where new elements should be added (spliced in).

The second parameter (0) defines how many elements should be removed.

The rest of the parameters ("Lemon" , "Kiwi") define the new elements to be added.

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.splice(2, 0, "Lemon", "Kiwi"); console.log(fruits); </script>

7) Merging (Concatenating) Arrays

The concat() method creates a new array by merging (concatenating) existing arrays.

Example:

<script> var myGirls = ["Cecilie", "Lone"]; var myBoys = ["Emil", "Tobias", "Linus"]; var myChildren = myGirls.concat(myBoys); // Concatenates (joins) myGirls and myBoys console.log(myChildren); </script>

8) Slicing an Array

The slice() method slices out a piece of an array into a new array.

Example:

<script> var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"]; var citrus = fruits.slice(1); console.log(citrus); </script>

9) Sorting an Array

The sort() method sorts an array alphabetically.

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.sort(); // Sorts the elements of fruits console.log(fruits); </script>

10) Reversing an Array

The reverse() method reverse an elements in an array.

Example:

<script> var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.reverse(); // reverse the order of the elements console.log(fruits); </script>
